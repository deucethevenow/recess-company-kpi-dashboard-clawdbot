## KPI Definitions

Document each KPI with:
- **Definition**: What the metric measures
- **Formula**: How it's calculated
- **Source**: Data tables/sources used
- **Refresh**: How often it's updated
- **Owner**: Who's responsible for the metric

## Dashboard Guide

For each dashboard, include:
- Purpose and audience
- Key metrics displayed
- Filters and interactions
- Data refresh schedule

## Adding New KPIs

1. Define the KPI in documentation first
2. Write the SQL query in `queries/`
3. Add to the appropriate dashboard
4. Test with sample data
5. Deploy and monitor
